      Description

      When there is connected the trainer with the cadence sensor and you are pedaling everything is fine until you stop pedaling, there is an N/A shown despite the fact that the cadence meter is still connected. There should be '0' cadence. N/A is signalizing that the connection is lost or the cadence meter is not connected.
